O **CodeCraft Summit** é uma aplicação interativa que gerencia inscrições para eventos técnicos. O grande diferencial deste projeto é a experiência do usuário fluida, onde a página se transforma após a inscrição sem a necessidade de recarregamento.

---

## 🛠️ Funcionalidades

- **Inscrição Dinâmica:** Captação de dados (E-mail e Telefone) com validação.
- **Sistema de Indicações (Referral):** Gera um link único para cada novo inscrito baseado em lógica de randomização.
- **Dashboard de Stats:** Feedback imediato sobre quantas pessoas se inscreveram através do link do usuário.
- **Single Page Logic:** Gerenciamento de "telas" (Início e Confirmação) inteiramente via JavaScript.

## 🧠 Conceitos de Engenharia Aplicados

Neste projeto, foquei em transformar o JavaScript no motor principal da interface:
1. **Manipulação de DOM:** Criação de templates literais dinâmicos para atualizar o `innerHTML` da aplicação.
2. **Data Handling:** Uso de `FormData` para extração limpa de dados de formulários e métodos de array como `find` e `filter` para gerenciar a lista de usuários em memória.
3. **Persistência Lógica:** Implementação de funções de busca e salvamento para simular o comportamento de uma base de dados real.

## 🚀 Tecnologias

- **JavaScript (ES6+):** Lógica core e gerenciamento de estado.
- **HTML5 & CSS3:** Interface responsiva e estilização moderna com foco em UX.
- **Node.js:** Ambiente de desenvolvimento.

---

## ⚙️ Como visualizar

1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Realize uma inscrição teste e veja o seu link de convite ser gerado instantaneamente!
